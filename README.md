# JobHub - Full-Stack Job Portal System

JobHub is a comprehensive, full-stack web application designed to seamlessly connect job seekers with recruiters. The platform is built using a robust Spring Boot backend and a modern, responsive HTML/CSS/JS frontend featuring a premium glassmorphism design.

## 🚀 Features

### Authentication & Security
- Secure role-based access control (Job Seeker, Recruiter).
- JWT (JSON Web Token) authentication.
- Passwords encrypted securely using BCrypt.

### For Job Seekers
- **Find Jobs:** Browse and search for the latest job postings.
- **Apply Instantly:** One-click application process.
- **Track Applications:** Dedicated dashboard to view applied jobs and track status (Applied, Shortlisted, Rejected).
- **Profile Management:** Update education, skills, and experience.
- **Resume Upload:** Upload standard PDF/DOCX resumes for recruiters to review.

### For Recruiters
- **Post Jobs:** Easily publish new job opportunities with details like Salary, Location, Skills, and Company Name.
- **Manage Postings:** View and delete active job postings.
- **Applicant Tracking:** View all applications for a specific job.
- **Evaluate Candidates:** Download applicant resumes and seamlessly accept (shortlist) or reject applications.

## 🛠️ Technology Stack

### Backend
- **Java 17** with **Spring Boot 3.x**
- **Spring Security** & **jjwt** (for JWT authentication)
- **Spring Data JPA** & **Hibernate**
- **MySQL Database**
- **Maven** for dependency management

### Frontend
- **HTML5** & **CSS3** (Custom Vanilla CSS featuring glassmorphism and modern UI elements)
- **Vanilla JavaScript (ES6)** (Fetch API for RESTful integration)
- **FontAwesome** for icons
- **Google Fonts** (Outfit)

## ⚙️ Setup and Installation

### Prerequisites
- **Java 17** installed.
- **MySQL Server** installed and running on port `3306`.
- A modern web browser.

### 1. Database Configuration
1. Open MySQL and create a database named `job_portal`:
   ```sql
   CREATE DATABASE job_portal;
   ```
2. The application is configured to connect to MySQL using the username `root` and password `root123`. 
   *(If your credentials differ, update them in `backend/src/main/resources/application.properties`)*.

### 2. Running the Backend
1. Open a terminal or command prompt.
2. Navigate to the `backend` directory:
   ```bash
   cd backend
   ```
3. Run the Spring Boot application using Maven:
   ```bash
   # On Windows
   .\mvnw.cmd spring-boot:run
   
   # On Mac/Linux
   ./mvnw spring-boot:run
   ```
4. The server will start on `http://localhost:8081`. 
   *(Note: The database tables will be automatically generated by Hibernate on startup).*

### 3. Running the Frontend
Because the frontend is built using pure HTML/CSS/JS, no Node.js or development server is required!

1. Navigate to the `frontend` directory in your File Explorer.
2. Double-click the `index.html` file to open it in your browser.
*(Alternatively, you can serve it via VS Code's "Live Server" extension for a smoother development experience).*

## 🤝 API Endpoints Reference

- **Auth**: `POST /api/auth/signup`, `POST /api/auth/signin`
- **Jobs**: `GET /api/jobs`, `POST /api/jobs`, `DELETE /api/jobs/{id}`
- **Applications**: `POST /api/applications/apply/{jobId}`, `GET /api/applications/my-applications`
- **Profiles**: `GET /api/users/profile`, `PUT /api/users/profile`, `POST /api/users/upload-resume`
